git push origin master(版本库推送)
git remote add origin git@server-name:path/repo-name.git(关联一个远程库)
git commit -m "wrote a readme file" (提交版本库的文件)
git add wenjian       (给github 版本库增加 记录)
git status  (查看状态)
git diff readme.txt (比较一个文件修改后的不同)
git log  (查看版本日志 便于回退版本)
git log --pretty=oneline (版本日志)
git reset --hard HEAD^  (版本回退)
git reset --hard 3628164 (版本前进 ，回退以后再回去)
git reflog       (对版本做了提交以后，然后想前进 先用这个找到日志 然后用上面的命令前进)
